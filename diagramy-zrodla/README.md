# Pliki źródłowe diagramów

Folder dla **edytowalnych źródeł diagramów UML** (poza obrazami wstawionymi do sprawozdania).

## Co tu trafia

Edytowalne pliki z programu w którym tworzycie diagramy, np.:
- Visual Paradigm — `*.vpp`
- Enterprise Architect — `*.eap`, `*.eapx`, `*.qea`
- StarUML — `*.mdj`
- draw.io / diagrams.net — `*.drawio`, `*.xml`
- PlantUML — `*.puml`
- Lucidchart — eksport XML

## Po co osobny folder?

Osoba odpowiedzialna za diagramy wrzuca tu pliki źródłowe → reszta zespołu może je otworzyć i poprawić, jeśli prowadzący poprosi o zmianę po prezentacji. Same obrazki (PNG/SVG) idą do folderów `TC_*`.
